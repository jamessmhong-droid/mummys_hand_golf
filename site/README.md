# 골프 경기력 핵심 연구 20선

골프 경기력(생체역학·퍼포먼스·트레이닝)에 관한 학술적으로 영향력 있는 논문 20편을 정리한 정적 웹사이트입니다.

- **V1 — 카드용 요약**: `golf-research-summary.html` (+ `pages/`) — 인스타그램 카드 업로드용 간결판
- **V2 — 전문가용 근거 자료집**: `v2/golf-research-v2.html` (+ `v2/pages/`) — 연구 설계·표본·통계·한계 포함
- 진입점: `index.html` (두 버전으로 연결)

## GitHub Pages 배포
1. 이 폴더의 내용을 저장소 루트(또는 `docs/`)에 업로드
2. GitHub 저장소 → **Settings → Pages**
3. Source를 **Deploy from a branch**, 브랜치 `main`(폴더 `/root` 또는 `/docs`)로 지정
4. 몇 분 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 확인

## 참고
- 인용 수는 데이터베이스·시점에 따라 달라지므로 순서는 대략적 영향력 기준입니다.
- V1 상단 이미지 자리는 이미지(누끼) 완료 후 교체 예정입니다.
