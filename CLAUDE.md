# mummys_hand_golf — 프로젝트 메모

골프 경기력 연구를 정리한 정적 HTML 사이트. GitHub Pages로 `site/`를 배포
(`.github/workflows/deploy-pages.yml`, `main` 푸시 시).

## 버전 구조
- **V1** — 인스타그램 카드용 요약 (`pages/`, `golf-research-summary.html`)
- **V2** — 전문가용 근거 자료집 (`v2/pages/`, `v2/golf-research-v2.html`)
- **V3** — 유튜브 원고 수준 롱폼 딥다이브 (`v3/`, `v3/index.html`)
- **V4** — 2013년 이후 최신 연구 트랙 (예정)
- 루트와 `site/`에 사본이 함께 존재하므로 편집 시 **양쪽 모두** 반영할 것.

## 콘텐츠 스타일 규칙
- **물리 용어 `work`는 한국어로 '일'이 아니라 '에너지'로 의역한다.** 직역 '일'이
  어색하기 때문. work = 클럽에 전달된 에너지(둘 다 단위 J)라 물리적으로도 타당.
  `power`는 '파워'(시간당 에너지)로 그대로 두어 구분을 유지한다. 학술 정확성을 위해
  원문 용어는 필요 시 '역학적 일(work)'로 한 번만 병기하고, 원제(영문 제목)는 보존.
